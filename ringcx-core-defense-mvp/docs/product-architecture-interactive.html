      技术架构

      当前实现是轻量浏览器产品：无构建依赖、单页运行、DOM/SVG 渲染、状态内存化、Telemetry 本地闭环。

      Runtime stack

      ClientHTML / CSS / vanilla JavaScript，当前发布体验可静态打开。
      RenderingDOM + SVG path 承载地图、站位、风险移动；视觉效果使用透明 PNG/WebP 资产。
      State单一 in-memory state 管理 wave、credit、Trust、SLA、defenders、enemies、mini-games。
      TelemetryTelemetry resolver 识别 local/staging/production、channel、share_id、data_scope。
      Local DataPython collector + SQLite run-history + local dashboard 支撑本地 PM 调优。
